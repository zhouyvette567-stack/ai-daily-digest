📖 项目简介

每日 AI 资讯日报是一个自动化的 AI 资讯聚合项目，每日自动抓取全球 AI 领域最新动态，经过 AI 整理后生成简洁易读的日报，无需人工干预。

🏷️ 内容分类

每日固定 10 条内容，分为四大类：

技术动态（2条） AI 论文（3条） 新产品（3条） 行业热点（2条）
  • 技术动态：GitHub 热门 AI 项目、开源工具更新
  • AI 论文：arXiv 最新人工智能学术论文
  • 新产品：Hugging Face 最新模型、AI 工具发布
  • 行业热点：AI 行业新闻、热点讨论

🔗 数据来源

🐙 GitHub Trending
📄 arXiv 论文库
🤗 Hugging Face
🔥 Hacker News

⏰ 更新频率

🤖 每日自动更新一次，通过 GitHub Actions 自动运行爬虫并部署

爬虫每日（UTC+8 早上 8:00）自动运行，抓取最新内容并更新网页，全程无需人工操作。

🛠️ 技术栈

  • 爬虫：Python（requests）
  • 前端：纯 HTML + CSS + JavaScript（无框架）
  • 自动化：GitHub Actions
  • 部署：GitHub Pages
  • 存储：JSON 文件（无数据库）
